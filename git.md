## Списки 

Чтобы создать ненумерованные списки необходимо пункты выделить (*). Или занком(+) Например, вот так:

* Элемент 1

* Элемент 2

* Элемент 3

+ Элемент 4

Чтобы создать нумерованные списки необходимо пункты пронумеровать. Например, вот так:

1. Пункт

2. Пункт

3. Пункт
